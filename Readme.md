Description:
The interaction model I am doing is a drag and drop interaction model. The player is given
blocks and a board-grid that they can drag and drop the blocks into. The objective of this 
interaction model is so I am able to create a block game. The block game will be a puzzle
type game that requires the player to put blocks on a board.

Write-up:
The way this interaction model will affect this game is simply everyway. This interaction 
model is crucial to the gameplay without this interaction model the player could not actually
play the game they would be able to strategize in a way. But they could not complete any
objects of the game.
