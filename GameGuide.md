1. Start the Game
Three block pieces appear below the board.
Each piece can be one of several shapes (e.g., 3×3 or 2×3).

2. Drag and Drop Blocks
Click or tap and drag a block onto the board.
The block will “snap” to the nearest valid position if it fits.
If it overlaps existing tiles or extends off the board, it returns to your hand.

3. Clearing Lines
Completing any full row or column clears those cells.
Cleared spaces become available again for new blocks.

4. Piece Usage
Each block can only be placed once.
After placing a block successfully, it disappears from your hand.

5. Fail Condition
The game ends when no remaining blocks can legally fit anywhere on the board.
The game also ends when all blocks have been placed and the board is not clear.

6. Saving and Loading
Use the Pause Menu to access Save and Load options.
Saving stores the current board and block positions to a JSON file.
Loading restores your saved game state exactly where you left off.

7. Restarting
The Restart button resets the board and blocks for a fresh start.

8. Pausing
Press Pause to freeze the game.
Inside the Pause Menu, you can Resume, Save, Load, or Restart.
