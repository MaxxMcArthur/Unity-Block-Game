# Unity Block Game

A block based game built in Unity using C#. This project focuses on implementing core gameplay mechanics, collision handling, and real time player interaction within the Unity game engine.

## Overview

This project is a single-game implementation developed in Unity. It demonstrates the use of C# scripting to control gameplay behavior, manage game state, and handle interactions between objects in a real time environment.

The game centers around block-based mechanics, showcasing movement, collisions, and game logic within a structured Unity project.

## Features

- Built using Unity game engine
- C# scripting for game logic
- Real-time gameplay interaction
- Collision detection and object interaction
- Modular component-based design using Unity architecture

## Tech Stack

- Unity
- C#

## Project Structure

```
Assets/
  Scripts/

```

## How to Run

1. Open the project in Unity Hub
2. Load the main scene
3. Press Play in the Unity Editor

## Concepts Demonstrated

- Unity component based architecture
- C# scripting for gameplay systems
- Collision detection and physics handling
- Game loop and real time updates
- Scene and object management

## Notes

This project focuses on implementing gameplay systems and mechanics rather than visual polish. It serves as a demonstration of Unity development fundamentals and interactive system design.
